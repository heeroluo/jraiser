Tabs ( options )

参数：

optionsObject
组件设置
[options.$wrapper]NodeList
组件容器。 如不指定，则tabs和panels参数必须为NodeList类型
[options.$tabs='.ui-tabs__$tabs__item']NodeList|String
选项卡节点。 可以是节点集合或者以wrapper为上下文的选择器
[options.$panels='.ui-tabs__$panels__item']NodeList|String
内容面板节点。 可以是节点集合或者以wrapper为上下文的选择器
[options.active=0]Number
初始激活项
[options.event='click']String
选项卡触发内容显示的事件
[options.useHashStorage=false]Boolean
是否通过锚点记录当前选项卡
[options.activeTabClass='ui-tabs__$tabs__item--active']String
选项卡在激活状态下的CSS类
[options.activePanelClass='ui-tabs__$panels__item--active']String
内容面板在激活状态下的CSS类
[options.activePanelStyle]Object
内容面板在激活状态下的样式，默认为display:block
[options.inactivePanelStyle]Object
内容面板在非激活状态下的样式，默认为display:none
[options.next]Function(current,total)
返回下一个序号的函数，默认为当前序号加一
[options.prev]Function(current,total)
返回上一个序号的函数，默认为当前序号减一
[options.playInterval=5000]Number
播放间隔

activate ( n )

激活某个选项卡

参数：

nNumber|String
选项卡序号或内容面板id

next ( )

激活下一个选项卡

prev ( )

激活上一个选项卡

play ( )

开始播放（逐个激活）选项卡

pause ( )

暂停播放选项卡

stop ( )

停止播放选项卡

beforeactivate ( e )

激活某个选项卡前触发

参数：

eObject
事件对象
e.sourceEventObject
源事件对象
e.$newTabNodeList
即将被激活的选项卡
e.$newPanelNodeList
即将被激活的内容面板
e.newActiveNumber
即将被激活项的数字索引
e.$oldTabNodeList
即将被反激活的选项卡
e.$oldPanelNodeList
即将被反激活的内容面板
e.oldActiveNumber
即将被反激活项的数字索引
e.preventDefaultFunction
调用此方法可阻止activate动作
