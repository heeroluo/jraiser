Infrastructure

  • base/1.2/base 本模块提供最基础、最核心的接口。
  • dom/1.2/dom 本模块提供选择器查询接口以及包含大量DOM操作接口的节点集合类型。
  • promise/1.2/promise 本模块对Promise编程模式提供支持。
  • pubsub/1.2/pubsub 本模块提供事件驱动机制。
  • tween/1.0/tween 本模块提供补间数值变化相关接口。

Utility

  • ajax/1.5/ajax 本模块提供Promise化的异步请求接口。
  • cookie/1.1/cookie 本模块提供Cookie读写接口。
  • crypto/1.1/md5 MD5加密算法。
  • crypto/1.1/sha1 SHA1加密算法。
  • querystring/1.1/querystring 本模块提供URL参数序列化与反序列化接口。
  • timespan/1.0/timespan 本模块提供时间跨度操作。
  • tmpl/2.2/tmpl 微型模版引擎。
  • ua/1.0/ua 本模块提供基于UserAgent的常用环境检测。
  • web-storage/1.1/web-storage 本模块提供本地存储接口的封装（不兼容IE6/7）。
  • xtpl/4.6/xtpl XTemplate模板引擎。
