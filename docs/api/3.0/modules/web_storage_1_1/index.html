web-storage@1.1

本模块提供本地存储接口的封装（不兼容IE6/7）。

  • 方法
  • 回到顶部

get ( key )

获取存储项。

参数：

keyString
键名。

返回值：

String存储值。

getAsJSON ( key )

获取存储项并转换成JSON对象。

参数：

keyString
键名。

返回值：

ObjectJSON对象。

remove ( key )

移除存储数据。

参数：

keyString
键名。

set ( key , value , [expires] )

存储数据。

参数：

keyString
键名。
valueAny
键值。其中Object类型（不包括其子类）会被序列化。
[expires]Date|Number|String
过期时间。 为日期类型时表示绝对时间； 为数字或字符串时表示相对时间（当前时间+相对值），支持格式同timespan模块。
