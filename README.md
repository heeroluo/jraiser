# JRaiser — 模块化的JS基础库

## 基础库
包含Web开发中最常用的基础功能模块，通用性强。配有API文档，模块功能一目了然。

## 模块化
使用基于CMD规范的 [Bowl.js](//github.com/heeroluo/bowljs) 作为加载器，模块的封装、调用更为方便和严谨，模块间的依赖管理也变得极为简单。

## 兼容性
大部分模块兼容PC浏览器（IE 6+）和移动浏览器（iOS 6.0+、Android 4.0+）；小部分模块在IE 6/7下不可用。
